# HR Employee Attrition Analysis

## 📊 Objective
Analyze employee data to understand factors affecting attrition.

## 🛠 Tools Used
- Python (Pandas)
- Power BI
- Excel

## 📌 Steps
- Data cleaning
- Exploratory Data Analysis (EDA)
- Visualization
- Dashboard creation

## 📈 Key Insights
- High attrition in certain departments
- Salary impact on retention
- Work-life balance effect
